# DialIQ 📞🤖

DialIQ is an AI-powered voice calling system that can interact with multiple users through human-like conversations, ask or answer questions, analyze context, and store structured insights in a database.

## Features
- AI-based voice interaction
- Human-like phone conversations
- Multi-user calling support
- Context analysis from conversations
- Automatic database entry
- Local prototype support (zero-cost setup)

## Tech Stack
- Python
- MySQL
- Local AI Models
- Speech-to-Text (STT)
- Text-to-Speech (TTS)

## Goal
Build a smart AI calling assistant that can automate conversations, collect information, and trigger actions based on call context.
